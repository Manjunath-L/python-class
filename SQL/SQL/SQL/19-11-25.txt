------  PRESENT IN OTHER TABLE THEN WE CAN GO WITH SUB-QUERY.

5. WQTD DETAILS OF EMP WHO GETTING COMM MORE THAN JAHNAVI AND YOUNGER THAN SAMEER?
----------------------------------------------------------------------------------
SELECT *
FROM EMP
WHERE COMM > (SELECT COMM 
                   FROM EMP 
                   WHERE FNAME = 'JAHNAVI') AND DOB>(SELECT DOB
                                                         FROM EMP
                                                          WHERE FNAME ='SAMEER');

TYPES OF SUB-QUERIES:
---------------------

1. SINGLE ROW SUB-QUERY: 
------------------------
IF THE INNER QUERY RETURN SINGLE ROW AS OUTPUT WE TREATED AS SINGLE ROW SUB-QUERY.

> WE CAN USE NORMAL OPERATORS (=, !=, >, <).

2. MULTI ROW SUB-QUERY:
-----------------------
IF THE INNER QUERY RETURN MULTIPLE ROWS AS OUTPUT WE TREATED AS MULTI ROW SUB-QUERY.    

> WE CAN USE ONLY SPECIAL OPERATOR(IN, LIKE, ANY, ALL).

12. WQTD DETAILS OF EMP WHO ARE DELIVERED THE PRODUCT TO THE CUSTOMER WHO BELONGS TO THE MUMBAI CITY?
-----------------------------------------------------------------------------------------------------
SELECT *
FROM EMP
WHERE EID IN (SELECT EID
              FROM ORDERS
              WHERE ORDER_ID IN (SELECT ORDER_ID
                                 FROM CUSTOMER
                                 WHERE LID IN (SELECT LID
                                               FROM LOCATION
                                               WHERE CITY = 'MUMBAI')));
