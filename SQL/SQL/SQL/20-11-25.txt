14. WQTD DETAILS OF EMP WHO ARE GETTING SAL MORE THAN ALL SALESMAN?
-------------------------------------------------------------------

SELECT *
FROM EMP
WHERE SAL > ALL(SELECT SAL 
             FROM EMP 
             WHERE JOB ='SALESMAN');

ALL:
---- 
    THIS IS SPECIAL OPERATOR IT WILL ACCEPT N NUMBER OF VALUES AT RHS AND SINGLE VALUE AT LHS.

SYNTAX:  COL_NAME/EXPRESIION >,<, ALL(V1,V2,V3,......);

IT WILL WORK BASED ON AND CONDITION.

15. WQTD DETAILS OF EMP WHO ARE GETTING SAL MORE THAN ANY SALESMAN?
-------------------------------------------------------------------

SELECT *
FROM EMP

             FROM EMP 
             WHERE JOB ='SALESMAN');

ANY:
----
SYNTAX:  COL_NAME/EXPRESIION >,<, ANY(V1,V2,V3,......);

IT WILL WORK BASED ON OR CONDITION.
