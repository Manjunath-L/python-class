SELECTION:
          IT IS A PROCESS TO SELECT RECORDS FROM THE TABLE BY USING COL_NAME/EXPRESSION BY PRIOVIDING

JOINS:
     TO RETRIVE THE DATA FROM MULTIPLE TABLE SIMULTANEOUSLY.



Order of Execution 
--------------------
> FROM
> SELECT 

FROM Clause 
-------------
> FROM Clause always execute first in query. It takes Table_name as argument. 
> It goes to database, search for the table and put the table under execution. 

SELECT Clause 
--------------
> SELECT Clause executes after FROM Clause. 
> It takes */Column_name/expressions as arguments. 
> SELECT Clause goes to the table which is under execution, gets the data and 
   display the output.
